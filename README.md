## Usage/Examples
Zbuduj projekt: 
```shell

$ mkdir build && cd build

$ cmake ..

$ cmake --build .


```
w folderze:
 ./build/bin
znajdują się pliki wykonywalne z testami jedostkowymi
przykładowy program: 
- arx_ut
- PSS_Lab
Aby testy jednostkowe:
```C++

./PSS_Lab/build/bin/arx_ut

```
Aby uruchomić program:
```C++

./PSS_Lab/build/bin/PSS_Lab

```
